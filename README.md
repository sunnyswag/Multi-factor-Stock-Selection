# auto_stocken
自动选股，自动交易

## 选股策略(短线)

```python
# 进场:
if 金叉 && MACD > -0.15:
    进场()

# 出场:
if 死叉:
    出场()
else：
    # 方案：使用前2，前1，前0天平均值进行测试
    
    昨天交易量平均值 = (前天交易量 + 昨天交易量) / 2
    今天交易量平均值 = (昨天交易量 + 今天交易量) / 2
    
    if 今天交易量平均值 >= 昨天交易量平均值:
        继续持有()
    else:
        准备出场()
        
def 准备出场():
    if 明天上涨:
        将状态改为进场
    else:
        出场()

```
